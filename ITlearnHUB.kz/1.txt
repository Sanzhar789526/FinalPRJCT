Главная страница:
Разделы популярных курсов, фильтрация по темам, отзывы, призывы к действию.
Каталог курсов:
Карточки курсов с фильтрацией по категориям, рейтингу и стоимости.
Поле поиска и сортировка по популярности и дате добавления.
Страница курса:
Описание, программа курса, преподаватель, отзывы, учебные материалы.
Функции просмотра уроков с возможностью продолжения, вопросы и комментарии.
Личный кабинет студента:
Список текущих и завершенных курсов, прогресс, доступ к пройденным материалам и сертификатам.
Возможность сдачи заданий и прохождения тестов.
Личный кабинет преподавателя:
Создание и редактирование курсов, загрузка материалов, планирование занятий.
Статистика по успеваемости студентов, отзывы и комментарии.
Создание и управление уроками и заданиями:
Возможность добавления видео, документов, тестов и контрольных вопросов.
Интерактивные задания и возможность тестирования для студентов.
Интерактивный модуль для проведения занятий:
Поддержка видео-конференций и чатов, виртуальная доска и аннотации для преподавателей.
Коммуникация между студентами и преподавателями:
Чаты или форумы для обсуждения курсов, возможность обратной связи от студентов.
Этап 4: Интеграция с бэкендом
Система управления ролями:
Обеспечьте разделение доступа для студентов и преподавателей.
Настройте API для безопасного хранения данных студентов и преподавателей, JWT для авторизации.
API для курсов и прогресса:
Реализуйте подключение к API для управления курсами, заданиями и получения данных по прогрессу студентов.
Интеграция системы оплаты:
Разработайте страницу оплаты и подключите платежные шлюзы, обеспечивая безопасность данных.
